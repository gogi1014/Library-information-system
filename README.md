Library Information System (Native PHP)
Technology Stack: PHP, MySQL, JavaScript (jQuery), Bootstrap.

Key Features
Advanced Search Engine: Implemented a multi-criteria search (by author, title, or tag) using SQL Prepared Statements to ensure high performance and security.

Dynamic Content Management: Full CRUD system for managing books and users, including a custom-built administrative interface.

User & Admin Management: Secure login system with session management and cookie-based authentication.

Interactive UI Elements: Integration of real-time search filtering on the front-end using JavaScript to improve user experience.

Review & Rating System: A feature-rich module allowing users to leave ratings and comments on books, complete with visual star-rating displays.
